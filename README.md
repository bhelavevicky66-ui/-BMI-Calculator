💚 VitalityCalc - BMI Calculator

VitalityCalc is a modern and user-friendly BMI (Body Mass Index) calculator that provides real-time health diagnostics. It helps users calculate their BMI and understand their health category such as Underweight, Normal, Overweight, or Obese.

🚀 Features

📊 Real-time BMI calculation

🎯 Accurate health classification

📱 Clean and responsive UI design


🧾 Input Fields

Users need to enter basic details:

Age

Gender (Male/Female)

Height (cm)

Weight (kg)

📈 Output

After clicking the Calculate button:

BMI score is displayed (example: 24.7)

Health status is shown:

🔵 Underweight

🟢 Normal

🟡 Overweight

🔴 Obese

A helpful message is also displayed
